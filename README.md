# Python Solutions to DDG (Keenan Crane)

## Overview
This repository contains partial Python implementations and solutions for exercises from book **DISCRETE DIFFERENTIAL GEOMETRY: AN APPLIED INTRODUCTION** by Keenan Crane.

It includes core DDG algorithms, mesh processing functions, and PyOpenGL-based 3D visualization for geometry rendering. The code can also be used as a lightweight Python toolkit for discrete differential geometry learning and development.

If you can read Chinese, [this article](https://zhuanlan.zhihu.com/p/2028207376482714165) introduces the core content of the book, paired with the code in this repository.

## Features
- Python-based DDG algorithm implementations
- Partial solutions to Keenan Crane's DDG course exercises
- 3D visualization powered by PyOpenGL
- Reusable modular toolkit for geometry processing
- Clean, readable code for learning and reference

## Usage
Run the Python scripts directly to test algorithms and visualization.

Import modules as a toolkit for your own DDG or geometry processing projects.

## Notes
This is a partial solution for personal study.

For the full DDG course and official materials, visit: [Keenan Crane / DDG](https://www.cs.cmu.edu/~kmcrane/Projects/DDG/)

#  DDG（Keenan Crane）的 Python 习题答案

- 模块化设计，可作为工具包直接使用

直接运行 Python 脚本即可测试算法与可视化效果。
